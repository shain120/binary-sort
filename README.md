# Binary Insertion Sort

以 Python 實作的 **二元插入排序（Binary Insertion Sort）**，改良自傳統插入排序，使用二元搜尋找插入位置。

## 原理

1. 從未排序區間取出一個元素
2. 用**二元搜尋**在已排序區間找正確插入位置
3. 將元素向右移動，騰出位置插入

## 複雜度

| | 標準插入排序 | 二元插入排序 |
|---|:---:|:---:|
| **比較次數** | O(n²) | O(n log n) |
| **移動次數** | O(n²) | O(n²) |
| **時間複雜度** | O(n²) | O(n²) |
| **空間複雜度** | O(1) | O(1) |

> 比較次數從 O(n²) 降至 O(n log n)，但移動次數不變。

## 使用方式

```bash
# 直接執行（互動模式）
python3 binary_search.py

# 互動輸入範例
# 請輸入數字（以空格分隔）：64 34 25 12 22 11 90
# 排序前：[64, 34, 25, 12, 22, 11, 90]
# 排序後：[11, 12, 22, 25, 34, 64, 90]
```

## 測試

執行時會自動跑完所有單元測試，支援：
- 一般亂序
- 已排序陣列
- 逆序陣列
- 含重複元素
- 單一元素
- 空陣列
- 負數與正數混合

## 核心函式

- `binary_search(arr, target, left, right)` — 二元搜尋
- `binary_insertion_sort(arr)` — 二元插入排序（原地排序）
